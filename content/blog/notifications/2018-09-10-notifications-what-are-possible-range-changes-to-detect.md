You can detect any possible value of a selected variable range change. You can select these ranges in the when case options while editing [notifications](f7277d70-7b35-489b-b378-009a690e0a3f).

<!-- more -->

*   Normal → warn
*   Normal → error
*   Normal → any
*   Warn → normal
*   Warn → error
*   Warn → any
*   Error → normal
*   Error → warn
*   Error → any
*   Any → normal
*   Any → warn
*   Any → error
*   Any → any
